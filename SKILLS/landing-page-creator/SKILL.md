# Landing Page Creator Skill

This skill provides architectural principles for creating high-converting landing pages. When tasked with designing, outlining, or building a landing page, ALWAYS structure it according to these sequential principles.

## Landing Page Architecture

### 1. HERO
- **Elements:** Eyebrow -> Headline -> Subheadline -> CTA -> Trust Signals
- **Job:** Get email or get scroll.

### 2. SUCCESS (conditional)
- **Elements:** Checkmark -> Confirmation -> Deliverable list
- **Job:** Kill buyer's remorse.

### 3. PROBLEM-AGITATE
- **Elements:** 3 problems with agitation -> Personal transition
- **Job:** Make status quo painful.

### 4. VALUE STACK
- **Elements:** 4 tiers descending -> Total value -> Your price
- **Job:** Make saying no feel stupid.

### 5. SOCIAL Proof
- **Elements:** Header -> 3 testimonials with specific results
- **Job:** Let others convince them.

### 6. TRANSFORMATION
- **Elements:** 4 stages: Quick win -> Compound -> Advantage -> 10x
- **Job:** Make outcome tangible.

### 7. SECONDARY CTA
- **Elements:** Avatar stack -> Question headline -> "Yes" button
- **Job:** Catch the scrollers.

### 8. FOOTER
- **Elements:** Logo -> Nav -> Legal -> Social
- **Job:** Professional legitimacy.
