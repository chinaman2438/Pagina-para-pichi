# Directives

This folder contains Standard Operating Procedures (SOPs) written in Markdown.

Each directive defines:
- **Goal** – What this workflow accomplishes
- **Inputs** – What the agent needs to get started
- **Tools/Scripts** – Which `execution/` scripts to use
- **Outputs** – What the deliverable looks like
- **Edge Cases** – Known issues, rate limits, gotchas

## Creating a Directive

Name your file descriptively: `verb_noun.md` (e.g., `scrape_website.md`, `generate_report.md`).

Directives are living documents — update them as you learn from real runs.
